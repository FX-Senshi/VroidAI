Public Web Publishing

This folder is ready for static hosting.

Recommended easiest options:
1. Netlify Drop
   Open https://app.netlify.com/drop
   Drag this publish-static folder into the page.
   Netlify will give you a public URL.

2. Cloudflare Pages Direct Upload
   Open https://dash.cloudflare.com/
   Pages -> Create project -> Direct Upload
   Upload this publish-static folder.

Notes:
- This viewer uses CDN-hosted three.js / three-vrm modules, so internet access is required.
- The public URL can be opened from smartphones over the internet.
